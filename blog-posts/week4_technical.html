Arrays and Hashes

First, what is an array and what is a hash?

An array, simply put, is a list of items, let's call them elements. You can make an array by using square brackets like this:

[red, yellow, green, blue]

You can retrieve elements from your array, you can ask how long you array is, see what the first item is... but these are just the basics.

So, when you create an array, it can be whatever you want, a mix of words and numbers (we call them strings and integers), thay can be in any order and you can even have duplicate elements.

Something interesting, each element in the array has an index value. The first element (red) is 0, the second (yellow) is 1, etc,.

array = ["red", "yellow", "green", "blue"]
array[2]
returns "green"

So we now know that arrays are indexed with numbers that start at 0, effectively, where is that item in the array -1.

A hash is a collection of key-value pairs.

hash = {
first_name => "Sarah"
last_name => "Showers"
age => 29
}

Values are assigned to keys using =>. First name is the key, was assigned it the value "Sarah". You can assign any Ruby object, see how age is an interger?

When we loop over an array, we say iterate. By using the .each method.
array = ["red", "yellow", "green", "blue"]

array.each { |a| puts "#{a}" }

The result is a list of colors:
red
yellow
green
blue

The hash will look more like this:
hash = {
first_name => "Sarah"
last_name => "Showers"
age => 29
}

hash.each { |x, y| puts "#{x}: #{y}"
The result is a list of key value pairs:
Sarah: first_name
Showers: last_name
29: age

What is the wierd stuff in curly brackets doing? Look at the array, again:
array = ["red", "yellow", "green", "blue"]
We tell the computer to take the array and for each element, print it to the console. We can use any placeholder for the name between the | | and puts puts each element on a new line.

For hashes we need two placeholders, one to represent each key and each value of the pair.
hash.each { |x, y| puts "#{x}: #{y}"
We loop through each item in the has and assign the key "x" and the value "y"
